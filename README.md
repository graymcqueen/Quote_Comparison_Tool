# YP Quote Comparison Tool

Branded producer tool for YourPolicy. Producer enters quote details, generates a polished client-facing PDF comparison.

## Features
- Producer input screen: client info, agent info, unlimited carriers
- Top 10 YP carriers pre-loaded (Progressive, Travelers, Liberty Mutual, PEMCO, Allstate, Foremost, Hanover, Goodville Mutual, Acuity, Hartford)
- Full coverage grid: BI, PD, UM, Collision, Comp, MedPay, Rental, Roadside, Dwelling, Contents, Loss of Use, Home Liability, Umbrella, and more
- Recommendation toggle — marks preferred carrier with star badge
- Auto-calculates savings vs. current carrier
- Branded YP client output — teal/gold color system, Nunito typography
- One-click Print / Save as PDF
- Railway-ready deployment

## Deploy to Railway
1. Push to GitHub
2. Railway → New Project → Deploy from GitHub repo
3. Auto-deploys — no config needed

## Local Dev
```bash
npm install
npm start
# Open http://localhost:3001
```
